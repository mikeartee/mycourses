# AWS Certified AI Practitioner (AIF-C01)

This course prepares you for the **AWS Certified AI Practitioner** exam (code **AIF-C01**), AWS's **foundational** certification for AI, machine learning, and generative AI. It's built for people who *use* AI, not necessarily build it, so there's **no coding required**. If you can follow a business scenario and reason about which AWS AI service fits, you're in the right place.

## The exam at a glance

| Fact | Detail |
|---|---|
| Exam code | AIF-C01 |
| Level | Foundational |
| Questions | 65 total (50 scored, 15 unscored) |
| Time | 90 minutes |
| Question types | Multiple choice, multiple response, ordering, matching |
| Scoring | Scaled 100-1000; **700** to pass |
| Cost | 100 USD |
| Validity | 3 years |

AWS uses a **compensatory** scoring model: you need to pass the exam overall, not each domain individually. Unanswered questions are marked wrong, so **never leave a blank**.

## Domains and weighting

| Domain | Focus | Weight |
|---|---|---|
| 1. Fundamentals of AI and ML | Core AI/ML concepts and the ML lifecycle | **20%** |
| 2. Fundamentals of Generative AI | Foundation models, tokens, prompts, use cases | **24%** |
| 3. Applications of Foundation Models | Prompt engineering, RAG, fine-tuning, model selection | **28%** |
| 4. Guidelines for Responsible AI | Fairness, bias, transparency, explainability | **14%** |
| 5. Security, Compliance, and Governance | Securing and governing AI solutions | **14%** |

Domains 2 and 3 together are **52%** of the exam, generative AI and foundation models (largely **Amazon Bedrock**) are the heart of AIF-C01. This course gives each domain its own module, plus an exam overview and an exam-tactics module.

## How to use this course

1. Start with the **Exam Overview** module for the format and the AWS AI service landscape.
2. Work through the five domain modules in order; each ends with a quiz.
3. Finish with **Exam Tactics**.
4. Get hands-on: try **Amazon Bedrock** and **PartyRock** in a free-tier account, playing with prompts teaches this faster than reading.
