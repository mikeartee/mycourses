# AWS Certified Solutions Architect - Associate (SAA-C03)

This course prepares you for the **AWS Certified Solutions Architect - Associate** exam (exam code **SAA-C03**). It's organised around the four official exam domains and the AWS Well-Architected Framework, the same lens the exam uses to judge a "good" architecture.

It assumes you know cloud basics and have some hands-on AWS exposure. It is a study aid, not a brain dump, expect to also do hands-on labs and timed practice exams.

## The exam at a glance

| Fact | Detail |
|---|---|
| Exam code | SAA-C03 |
| Questions | 65 total (50 scored, 15 unscored) |
| Time | 130 minutes |
| Question types | Multiple choice (one correct) and multiple response (two or more correct) |
| Scoring | Scaled 100-1000; **720** to pass |
| Cost | 150 USD (may vary by region) |
| Validity | 3 years |

There is **no partial credit** on multiple-response questions, you must select every correct answer and no incorrect ones.

## Domains and weighting

| Domain | Focus | Weight |
|---|---|---|
| 1. Design Secure Architectures | Identity, access, data protection, network security | **30%** |
| 2. Design Resilient Architectures | High availability, decoupling, fault tolerance, DR | **26%** |
| 3. Design High-Performing Architectures | Scalable compute, storage, databases, networking | **24%** |
| 4. Design Cost-Optimized Architectures | Right-sizing, pricing models, cost controls | **20%** |

Questions are drawn randomly, so any single exam varies, but over many questions the weighting holds. **Security is the single heaviest domain**, so it earns the most study time. This course puts one module on each domain, plus an overview and an exam-tactics module.

## How to use this course

1. Start with the **Exam Overview** module to understand the format and the Well-Architected lens.
2. Work through the four domain modules in order. Each ends with a quiz.
3. Finish with **Exam Tactics** for scenario-reading technique and a study plan.
4. Alongside this, build things in a real AWS account and take full-length timed practice exams.
