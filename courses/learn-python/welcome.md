# Learn Python: From Zero to Confident

Python is one of the best first languages to learn. It reads almost like English, it runs nearly everywhere, and it powers everything from small scripts to data science, web backends, and automation.

This course assumes **no prior programming experience**. You'll write real code from the first module.

## What you'll be able to do by the end

- Install Python and run code, both interactively and from a file
- Use variables and Python's core types (numbers, strings, booleans)
- Make decisions with `if`/`elif`/`else` and repeat work with loops
- Store and shape collections of data with lists, tuples, dictionaries, and sets
- Handle text confidently with string methods and f-strings
- Package logic into reusable functions and pull in code from the standard library

## How to use this course

Work through the modules in order. Each lesson has short code examples, type them out and run them, don't just read. Every module ends with a quiz so you can check what stuck.

## Course map

1. **Getting Started with Python** - what it is, running it, variables and types
2. **Making Decisions and Repeating Work** - conditionals, loops, comprehensions
3. **Working with Data** - lists, tuples, dictionaries, sets, and strings
4. **Organising Code with Functions** - functions, arguments, modules and imports
