# What Python Is and Why It's Worth Learning

Python is a **general-purpose programming language**. "General-purpose" means it isn't tied to one job, people use it for web apps, data analysis, machine learning, scripting, automation, games, and more.

## Why beginners start with Python

- **Readable.** Python code looks close to plain English, so you spend less time fighting punctuation and more time learning ideas.
- **Batteries included.** It ships with a large standard library, so a lot of common work (dates, files, web requests, JSON) is already available.
- **Everywhere.** It runs on Windows, macOS, and Linux, and it's one of the most in-demand languages in industry.
- **Huge community.** Almost any question you have was already answered somewhere.

## What "running code" actually means

A program is a list of instructions. You write those instructions as text, and the Python **interpreter** reads them top to bottom and does what they say.

Here is a complete Python program:

```python
print("Hello, world!")
```

`print(...)` displays whatever is inside the parentheses. Run that line and you'll see:

```
Hello, world!
```

That's it, one line is a real program.

## A taste of what's ahead

Don't worry about understanding all of this yet, just notice how little ceremony there is:

```python
name = "Ada"
age = 36

if age >= 18:
    print(name, "is an adult")
else:
    print(name, "is a minor")
```

No boilerplate, no semicolons, no type declarations. You'll learn every piece of this over the next few lessons.

## Key takeaways

- Python is a readable, general-purpose language used across many fields.
- A program is instructions the interpreter runs top to bottom.
- `print()` shows output.
